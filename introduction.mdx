# Python Port Scanner

A fast and lightweight **port scanner written in Python** that scans a target host for open TCP ports. Designed for learning network security, socket programming, and multithreading — with no external dependencies required.

<Warning>
  This tool is intended **for educational and authorized security testing only**. Never scan systems without explicit permission from the owner.
</Warning>

## Overview

The Python Port Scanner connects to each port in the specified range using TCP sockets, reports whether each port is open, closed, or filtered, and optionally saves results to a file. It uses Python's built-in `threading` module for parallel scanning and `argparse` for a clean command-line interface.

## How it works

The scanner resolves the target hostname to an IP address, then spawns one thread per port in the configured range. Each thread:

1. Enforces the rate limit by waiting if too many probes have been sent recently
2. Acquires a semaphore slot to cap the maximum number of simultaneous connections
3. Attempts a TCP connection with a 3-second timeout
4. Reports the port as **open**, **closed**, or **filtered** based on the result
5. Writes the result to a shared list (protected by a lock)

After all threads complete, results are sorted by port number and written to the output file.

## Requirements

- Python **3.8** or later
- No external packages — uses only the Python standard library (`socket`, `threading`, `argparse`, `json`, `csv`, `os`, `collections`, `datetime`)
